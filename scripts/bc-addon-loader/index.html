Instructions how to run Fantastic Ultimate Solution to Addon Management

To run FUSAM, choose ONE of the mentioned methods.
All of them will always start the latest FUSAM version without any need to change something on your side.
FUSAM is open source.

(A) Automatic loader:

1) Install Violentmonkey or a compatible userscript runner
2) Open following page and confirm script installation: link, alternatively enter the link in "Install from URL" option on violentmonkey's dashboard.

(B) Manual bookmark loader (Desktop):

1) Drag the following link to your bookmark bar: Bondage Club Addon Loader
2) Click the bookmark while in Club (or login screen)

(B) Manual bookmark loader (Mobile: Android):

1) Copy everything from the following text field:

2) Press 3 dots -> Bookmark star -> Edit (on bottom of the screen in popup)
3) Long-tap text in URL (or select all) and paste copied text
4) Leave bookmark menu using back arrow in top left corner

To activate:
1) Enter Club
2) Tap URL bar
3) Write the name of the bookmark you saved
4) Tap it (It will have star on the left)
*) Do not try selecting the bookmark from list of bookmarks, it won't work!

(B) Manual bookmark loader (Mobile: iOS):

1) Copy everything from the following text field:

2) Press "Share" -> "Add Bookmark" and Save
3) Open Bookmarks list -> press "Edit" (on the bottom of the screen in the popup)
4) Select the bookmark you just saved
5) [if applicable] Select "More" -> "Edit bookmark"
6) Hit "x" next to the URL to clear it and paste the copied URL
7) Leave the bookmark menu using "Done" in the top right corner

To activate:
1) Enter Club
2) Open Bookmarks list
3) Tap the bookmark you saved
